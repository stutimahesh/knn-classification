# 🤖 K-Nearest Neighbors (KNN) Classification

A simple Machine Learning project that demonstrates the **K-Nearest Neighbors (KNN)** algorithm on a **synthetically generated dataset**. The model is trained on labeled samples, predicts labels for unseen data, and compares the performance for different values of **k**.

---

## ✨ Features

- Generates a synthetic dataset using NumPy
- Creates labels based on a threshold value
- Trains a K-Nearest Neighbors (KNN) classifier
- Predicts labels for unlabeled samples
- Compares accuracy for multiple values of **k**
- Visualizes feature distribution using histograms

---

## 🛠️ Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- Jupyter Notebook

---

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/knn-classification.git
```

Install the required libraries:

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

Open the notebook and run all cells:

```bash
jupyter notebook KNN.ipynb
```

## 📈 Output
